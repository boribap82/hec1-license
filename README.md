# hec1-license

HEC1-WR 베타 배포본의 실행 인증 상태 파일.

- 앱이 기동 시 `license.json`(raw)을 읽고, **GitHub 응답의 서버 시각** 기준으로 유효기간을 판정한다 (사용자 PC 시계 조작 무의미).
- 인터넷 불통이면 앱은 실행되지 않는다 (베타 정책).
- 이 저장소에는 키의 SHA-256 해시와 만료일만 있다 — 민감정보 없음.

## 관리

| 하고 싶은 것 | 방법 |
|---|---|
| 전체 차단 (킬스위치) | 해당 키의 `"enabled": false` 로 수정 후 push |
| 유효기간 연장/단축 | `"expires"` 날짜 수정 후 push |
| 전체 공지 띄우기 | `"message"` 에 문구 입력 → 앱 기동 시 안내창 표시 |

반영 시간: push 후 raw CDN 캐시(약 5분) 이내.

## 키 등록

새 키 등록 시: `sha256(키 문자열)` 해시를 `keys`에 추가.
키 원문은 이 저장소에 올리지 않는다 (로컬 전용 — `D:\200_project\03-HEC1-WS\07-license\` 작업 메모 참조).
